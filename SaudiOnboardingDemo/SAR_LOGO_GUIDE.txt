──────────────────────────────────────────

✅ الحل:
Solution:

1. تأكد من اسم الصورة: sar_logo.png
   Check image name: sar_logo.png

2. تأكد من المكان الصحيح
   Check correct location

3. تأكد من تنسيق PNG
   Make sure it's PNG format

4. أعد بناء المشروع:
   Rebuild project:
   Cmd + Shift + K (Clean)
   Cmd + R (Run)


❌ المشكلة: الشعار صغير جداً أو كبير جداً
Problem: Logo too small or too big
──────────────────────────────────────────

✅ الحل:
Solution:

افتح CurrencyView.swift وغيّر الأحجام:
Open CurrencyView.swift and change sizes:

// للأماكن الكبيرة (مثل الرصيد الكلي)
// For large places (like Total Balance)
logoSize: 36  → غيّره لـ 40 أو 32

// للأماكن المتوسطة
// For medium places
logoSize: 18  → غيّره لـ 20 أو 16

// للأماكن الصغيرة
// For small places
logoSize: 14  → غيّره لـ 16 أو 12
